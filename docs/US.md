# US (User Stories & Acceptance Criteria)

## 1. 사용자 페르소나
- **일반 회원:** 글 쓰기, 댓글, 투표 참여를 원하는 커뮤니티 사용자
- **운영자:** 공지/게시글/일정/투표/사용자 관리를 운영하는 관리자
- **방문자:** 계정을 만들지 않았거나 미로그인 상태인 사용자

## 2. 사용자 스토리

### 2.1 공지사항
- **US-101 (방문자):** 공지 목록을 확인하고 원하는 공지를 클릭한다.  
  - **인수조건:** 목록에서 제목/등록일/고정 여부를 확인하고 상세 페이지로 이동 가능하다.
- **US-102 (운영자):** 중요 공지를 상단 고정한다.  
  - **인수조건:** 상단 고정 글이 목록 첫머리에 항상 노출되고, 해제 시 원래 정렬로 복귀한다.

### 2.2 소통 공간
- **US-201 (회원):** 자유게시판에서 글을 등록한다.  
  - **인수조건:** 제목/본문/이미지 첨부 후 저장되며, 목록에서 최신순으로 조회된다.
- **US-202 (회원):** 댓글/대댓글을 작성한다.  
  - **인수조건:** 부모 댓글과 연결되어 표시되고, 작성 후 즉시 목록에 반영된다.
- **US-203 (회원):** 좋아요를 누른다/취소한다.  
  - **인수조건:** 동시 요청에도 1인 1회 반영되어야 하며, 카운트가 일관되게 변한다.
- **US-204 (방문자):** 글을 조회한다.  
  - **인수조건:** 댓글/좋아요 버튼은 로그인 유도 상태로 노출된다.

### 2.3 주요 일정
- **US-301 (운영자):** 월간 일정을 등록한다.  
  - **인수조건:** 시작일/종료일/카테고리/내용이 저장되며 월간 캘린더에 표시된다.
- **US-302 (회원):** 다가오는 일정을 확인한다.  
  - **인수조건:** 날짜별 필터/카테고리 필터 적용 시 목록 반영된다.

### 2.4 투표/설문
- **US-401 (운영자):** 설문을 만든다.  
  - **인수조건:** 질문/옵션/복수선택/마감일/결과 공개 설정이 가능해야 한다.
- **US-402 (회원):** 투표에 참여한다.  
  - **인수조건:** 중복 참여가 차단되고, 마감 이후에는 추가 투표가 불가능하다.
- **US-403 (회원):** 결과를 확인한다.  
  - **인수조건:** 결과 공개인 경우 실시간 집계를 확인한다.

### 2.5 인증/프로필
- **US-501 (방문자):** 계정을 생성한다.  
  - **인수조건:** 이메일 중복 검사와 비밀번호 정책(최소 길이, 강도) 적용 후 가입 완료.
- **US-502 (회원):** 로그인 후 세션 유지한다.  
  - **인수조건:** 브라우저 재방문 시 자동 로그인을 제어할 수 있다.
- **US-503 (회원):** 프로필을 수정한다.  
  - **인수조건:** 표시명/소개/아바타를 변경하고 즉시 반영된다.

### 2.6 관리자 대시보드
- **US-601 (운영자):** 부적절한 게시글을 조치한다.  
  - **인수조건:** 숨김/삭제/경고를 선택할 수 있고 처리 이력이 남는다.
- **US-602 (운영자):** 활동 통계를 확인한다.  
  - **인수조건:** 기간별 게시글/댓글 수, 좋아요 수, 투표 참여율 집계 표시.

## 3. 비기능 인수조건
- 권한 오용: 권한 없는 요청은 403 또는 적절한 에러 응답
- 보안: 비밀번호, 세션, 파일 업로드에 대한 정책 로그 기록
- 성능: 목록 조회 95퍼센타일 지연시간 500ms 이하(목표)
